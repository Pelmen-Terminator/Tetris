# Тетрис — веб-версия

Классический Тетрис на чистом JavaScript с генерацией звуков.

## Управление

| Клавиша | Действие |
|---------|----------|
| ← / → | Движение |
| ↑ | Поворот |
| ↓ | Мягкое падение |
| Пробел | Сброс вниз |
| C | Холд |
| P | Пауза |

**Мобильные устройства:**
- Свайп ← → — движение
- Свайп вниз — сброс
- Тап — поворот

## Файлы

- `index.html` — разметка
- `style.css` — стили
- `script.js` — вся логика + звуки
- `korobeiniki_loop.mp3` — музыка (опционально)
